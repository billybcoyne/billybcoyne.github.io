Bootcamp 2021: Interactive shapes and colours

My first programming project, concluding the 2 week Programming Bootcamp at City (2021). This Processing project mostly demonstrates a comprehensive understanding of the use of interaction and methods, with the manipulation of variables. In the last 4 days ofthe Bootcamp, I created a program which draws shapes such as rectangles, circles and triangles depending on the users mouse position on the screen and whether key 'r', 'c' or 't' are held - drawing their respective shape. Additionally, I have added a colour wheel and a piece of code which allows me to obtain the colour of the pixel on the screen of my mouse postition whenever I right click. My next shape drawn will then be filled in with that corresponding colour. Clicking the 'l' key will clear all the drawings on the screen. All instructions for the program are printed in the Processing console. I would have loved to extend this project further by adding the introduction of groups/classes and the use of arrays, however time was short to get a final product. Additionally, the mathematical side of the coding which mostly included the drawing of the shapes took me a lot of time to produce, and I couldn't get the code for the triangles to work in the intended way. I would like to improve on this project in my spare time, and see what more I can do.
